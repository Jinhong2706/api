# 2048

经典数字合成游戏。在 4×4 棋盘上滑动方块，相同数字相遇就会合并成更大的数——目标是合成出 **2048**。

## 怎么玩

- 每次滑动，所有方块会朝该方向靠拢。
- **两个相同数字**撞在一起时合并成它们的两倍（例如 2 + 2 → 4，4 + 4 → 8）。
- 每次成功移动后，会在空位随机出现一个新方块（多为 **2**，偶尔是 **4**）。
- 棋盘被填满且无法再合并时，游戏结束。
- 合成出 **2048** 即达成目标；也可以选择继续玩，冲击更大的数字。

侧栏的 **历史最佳** 记录的是你曾达到过的**最高方块数字**（本机保存，刷新不丢）。

## 操作方式

### 电脑端

| 操作 | 按键 / 方式 |
|------|-------------|
| 上 / 下 / 左 / 右 | 方向键，或 **W A S D** |
| 新游戏 | 点击「新游戏」按钮 |

### 手机 / 触屏

| 操作 | 方式 |
|------|------|
| 移动 | 在屏幕上**滑动**，或点击右侧**方向按钮** |
| 新游戏 | 点击「新游戏」按钮 |

> 提示：只有方块实际发生移动或合并时，才会生成新方块。空滑无效。

## 胜利与结束

### 达到 2048

弹出恭喜提示后，你可以：

- **继续游戏**：挑战更高数字（4096、8192…）
- **新游戏**：清空棋盘，重新开局

### 游戏结束

当没有空位、也没有任何相邻相同数字可合并时，本局结束。点击「新游戏」再战。

## 小技巧

1. **尽量固定一个方向**堆叠大数字（很多人习惯把最大块放在某一角）。
2. 优先把相同数字排在同一行或同一列，方便连续合并。
3. 少让棋盘过早被小数字填满，留出操作空间。
4. 一次合并不够时，可以规划「连锁」：4 → 8 → 16 一路合成上去。

## 开始一局

1. 打开页面即可开玩（开局会随机出现两个方块）  
2. 用方向键、WASD、滑动或方向按钮移动  
3. 冲向 2048，刷新你的历史最佳  
4. 卡住了？点「新游戏」再来一轮  

祝你合成顺利，数字越滚越大 🔢
