# 贪吃蛇

经典贪吃蛇，开箱即玩。操控一条小蛇在地图上穿梭，吃掉食物、越变越长，同时别撞墙、别咬到自己。

## 怎么玩

- 小蛇会自动向前移动，你只需要控制方向。
- 吃到食物得 **10 分**，身体会变长一格。
- 撞到边界或撞到自己的身体，游戏结束。
- 目标很简单：在你选的速度下，尽量拿更高分。

## 操作方式

### 电脑端

| 操作 | 按键 / 方式 |
|------|-------------|
| 移动 | 方向键，或 **W A S D** |
| 开始 / 暂停 / 继续 | **空格键**，或点击游戏画面 |
| 重新开始 | 点击「重新开始」按钮 |

### 手机 / 触屏

| 操作 | 方式 |
|------|------|
| 移动 | 在画面上**滑动**，或使用底部**虚拟方向键** |
| 开始 / 暂停 / 继续 | **轻点**游戏画面 |
| 显示 / 隐藏方向键 | 点击侧栏「显示方向键 / 隐藏方向键」 |

> 提示：不能直接掉头（例如正在向右时不能立刻向左），避免一不小心咬到自己。

## 速度模式

开局前（或暂停 / 结束后）可以切换四种速度，**进行中无法切换**：

| 模式 | 手感 |
|------|------|
| 慢速 | 节奏宽松，适合熟悉操作 |
| 正常 | 默认速度，平衡手感 |
| 快速 | 反应要跟上 |
| 极速 | 极限挑战 |

每种速度会**单独记录最高分**。刷新页面后最高分仍会保留在本机。

## 小功能

- **暂停**：随时喘口气，再继续冲高分。
- **背景色**：侧栏可选多种深色主题（经典黑、深邃蓝、暗夜绿等），换个氛围再战。
- **新纪录**：打破当前速度下的最高分时，结算界面会显示「新纪录」。

## 开始一局

1. （可选）选好速度和背景色  
2. 点击「开始游戏」，或按空格 / 点画面  
3. 控制方向吃食物，活得越久、分越高  
4. 结束了？再来一局，冲更高纪录  

祝你玩得开心，蛇越长越好 🐍
